== Description ==

This plugin uses [Contact Form 7](https://ja.wordpress.org/plugins/contact-form-7/) to register information in the target database of [WiLL Mail](https://willcloud.jp/).

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wp-willmail-put` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress.
